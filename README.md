# mysql_jdbc

#### 项目介绍
使用最原始的JDBC连接mysql数据库的Demo、并对JDBC进行增删改查封装，多表插入时有一张表插入错误数据回滚封装。动态代理等。适合新手学习，能更加的了解持久框架的原理。包含数据库建表文件。
环境：jdk1.7 + myeclipse +mysql
